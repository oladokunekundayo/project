Course Description:

This course will introduce concepts in programming web applications and a variety of modern web application development tools, protocols, and technologies. Through a series of homework assignments, cadets will learn how to use such tools, protocols and technologies, and build an increasingly sophisticated client side web application using those. In later part of the semester cadets will do a large final project, in which cadets will design and implement a substantially dynamic and interactive web application as part of a project team. At the conclusion of the project cadets will demonstrate their web application to all classmates. Since the server-side web application will be covered in CIS 411 - Advanced Web Design, this course will mainly focus on client side web design and development.
